PROJECT TITLE: 
Breakout

PURPOSE OF PROJECT: 
An implementation of the Breakout game.  

VERSION or DATE: 
1.0

HOW TO START THIS PROJECT: 
Compile all the classes and press the run button.

AUTHORS: 
Barbara Ericson of Georgia Tech (based on work by Eric Roberts which was
a nifty assignment at SIGCSE 2006 but implemented using the Java Task Force
libraries instead)

USER INSTRUCTIONS: 
Use the left and right arrow keys to move the paddle.  Try to keep the ball
from getting past the paddle.  You have a total of 3 balls to use to get rid 
of all the bricks.  If you get rid of all of the bricks you win!
